# Spritesheet Metadata
JSON format.

Example metadata below for a spritesheet containing two 16x16 sprites with no border:
```JSON
{
	"border_size": 0,
	"sprites": [
		{
			"name": "verbose_image_name_1.png",
			"pos_x": 0,
			"pos_y": 0,
			"width": 16,
			"height": 16
		},
		{
			"name": "verbose_image_name_2.png",
			"pos_x": 17,
			"pos_y": 0,
			"width": 16,
			"height": 16
		}
	]
}
```
